# Telegram Bot - kbot

Цей проект реалізує простого Telegram-бота на мові Golang.

## Встановлення

1. Встановіть Golang.
2. Клонуйте репозиторій.
```bash
https://github.com/masteine/kbot.git
```

3. Встановіть залежності:
    ```bash
    go get github.com/spf13/cobra
    go get gopkg.in/telebot.v3
    ```
4. Отримайте токен бота через BotFather та додайте його до змінної середовища `TELE_TOKEN`.

## Запуск

```bash
go run main.go
```
